This module contains core libraries/engine of JDUT.

# Testing

* `logging.level.jdut` - Value for all of the logging level under `guru.mikelue.jdut`
* `logging.level.jdut.core` - Default for all of the logging level under `guru.mikelue.jdut.core`

```bash
mvn test -Dlogging.level.jdut.core=debug
```
