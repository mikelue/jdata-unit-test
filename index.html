jdata-unit-test(JDUT)

Table of Contents
  • Introduction
  • Install
    • Dependency setting
  • References

A framework for testing relational databases.

This framework is mainly inspired by DbUnit.

Introduction

JDUT currently contains two modules:

  • Core module - Using JDUT by Java API(including YAML loading)

  • TestNG module - Using JDUT by listeners of TestNG

  • JUnit4 module - Using JDUT by listeners of JUnit 4

Install

Currently, this framework is on SNAPSHOT of oss.sonatype.org.

You should put following setting of repository:

<project>
	<!-- ... Maven POM ... -->

	<repositories>
		<repository>
			<id>oss.sonatype.org</id>
			<name>oss.sonatype.org</name>
			<snapshots>
				<enabled>true</enabled>
				<checksumPolicy>fail</checksumPolicy>
			</snapshots>
			<releases>
				<enabled>false</enabled>
			</releases>
			<url>https://oss.sonatype.org/content/repositories/snapshots</url>
			<layout>default</layout>
		</repository>
	</repositories>
</project>

Dependency setting

Core Module
<dependency>
	<groupId>guru.mikelue.jdut</groupId>
	<artifactId>core</artifactId>
	<version>1.0.0-SNAPSHOT</version>
	<scope>test</scope>
</dependency>
TestNG Module
<dependency>
	<groupId>guru.mikelue.jdut</groupId>
	<artifactId>testng</artifactId>
	<version>1.0.0-SNAPSHOT</version>
	<scope>test</scope>
</dependency>
JUnit 4 Module
<dependency>
	<groupId>guru.mikelue.jdut</groupId>
	<artifactId>junit4</artifactId>
	<version>1.0.0-SNAPSHOT</version>
	<scope>test</scope>
</dependency>
